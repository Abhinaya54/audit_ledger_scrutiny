# Usage

Quick start (one line): create a venv, install deps, open the notebook, and run cells.

Notebook
- Open `src/notebook.ipynb` with JupyterLab or Jupyter Notebook.
- Select a cell and press Shift+Enter to run it step-by-step.

Command-line
- Install dependencies:

```powershell
pip install -r requirements.txt
```

- Run the generator script:

```powershell
python src\generator.py
```

Files
- Input sample: `src/data/synthetic_gl_2025.csv`
- Outputs and exports are written to the `data/` folder.
